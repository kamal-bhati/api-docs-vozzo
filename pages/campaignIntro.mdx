# Campaign Management API

The Campaign Management API allows you to create, manage, and monitor automated voice call campaigns using Vozzo's AI agents.

## Overview

- **Create Campaigns**: Schedule voice calls to multiple recipients with personalized messaging
- **Monitor Progress**: Track campaign status and individual call results  
- **Manage Calls**: View scheduled calls and their current status

## Key Features

- Bulk campaign creation with personalized metadata
- Flexible scheduling with preferred time slots
- Comprehensive reporting and analytics
- Retry mechanisms for failed call attempts

## Authentication
All campaign endpoints require Bearer token authentication.
