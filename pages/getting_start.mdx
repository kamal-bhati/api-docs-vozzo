## ðŸš€ Quick Start

### Prerequisites
- Vozzo AI Labs account
- API Key (`VOZZO_API_TOKEN`)
- AI Agent configured in your dashboard

### Basic Flow
1. **Authenticate** using your API Key (`Authorization: token <VOZZO_API_TOKEN>`)
2. **Create Campaign** with desired configuration (`POST /v1alpha/campaign`)
3. **Upload Contact Data** as part of campaign creation
4. **Monitor Progress** through campaign endpoints (`GET /v1alpha/campaigns`, `GET /v1alpha/campaign/{campaign_id}`)
5. **Analyze Results** with comprehensive reporting

## ðŸ“‹ API Features

### Campaign Management
- Create new campaigns (`POST /v1alpha/campaign`)
- List and filter existing campaigns (`GET /v1alpha/campaigns`)
- Retrieve detailed campaign information (`GET /v1alpha/campaign/{campaign_id}`)
- Monitor campaign status and metrics

### Scheduled Calls
- View all scheduled call items per campaign (`GET /v1alpha/campaign/{campaign_id}/items`)
- Filter calls by status (scheduled, in progress, completed, etc.)
- Get detailed call information and outcomes (`GET /v1alpha/campaign/{campaign_id}/item/{item_id}`)
- Manage call retries and follow-ups

### Authentication & Security
- API key management with scoped permissions
- API Key authentication (`Authorization: token <VOZZO_API_TOKEN>`)
- Role-based access control
- Audit logging for compliance
