# Scheduled Calls API

Manage and monitor individual scheduled calls within your campaigns.

## Overview

This API provides detailed visibility into each scheduled call, including:
- Current call status and history
- Call results and recordings
- Attempt logs and failure reasons
- Real-time call metrics

## Use Cases

- Monitor call progress in real-time
- Troubleshoot failed call attempts
- Analyze call performance and metrics
- Retrieve call recordings and summaries
