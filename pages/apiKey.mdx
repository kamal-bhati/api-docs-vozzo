### Steps to Get API Key:
1. Click the button above or visit [Vozzo Platform](https://platform.dev.vozzo.ai/my-agents)
2. Log in to your account
3. Navigate to your agent settings
4. Generate a new API key
5. Copy and securely store your API key

> **Important**: API keys provide full access to your account. Keep them secure and never share in public repositories.

### Using Your API Key
Once you have your API key, include it in the `Authorization` header of your requests:
